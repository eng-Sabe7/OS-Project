# OS-Project
Simple Shell &amp; Mini-FAT (Python) for OS (CS321), Assiut Univ. A stable shell that executes commands and simulates a FAT-based virtual disk with directories/files (create, delete, browse). Commands: cd, dir, copy, del, md, rd, rename, type, import/export. Run: cd OS_Project &amp;&amp; python main.py. Author: Ahmad Sobeh ([ahmadaymansobeh@gmail.com])
